Suppression de compte

Méthode pour supprimer son compte

  • Se rendre dans le menu situé en haut à droite
  • Appuyer sur le bouton "Profil"
  • Appuyer sur le bouton "Supprimer mon compte"
  • Une popup devrait appraitre. Cliquez à nouveau sur "Supprimer mon compte"
  • Votre compte devrait être supprimé. Si jamais vous n'y arrivez pas vous pouvez me contacter à : pierre.guillard.dev@gmail.com
